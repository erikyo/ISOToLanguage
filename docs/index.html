ISO To Language

Light and fast module to convert iso country and languages codes into country names, there is no unnecessary stuff only the bare essentials.

https://www.npmjs.com/package/isotolanguage

Fn: isValidIso

a function that formats the language and country into a single string. In this case will return false because EN is not a country code it is a language code

Example: isValidIso('EN')

Args: {"iso":"EN"}

false

Fn: format

a function that formats the language and country into a single string. defaults to locale format

Example: format('en', 'us')

Args: {"language":"en","country":"us"}

en_US

Fn: format (specifying a separator)

You can pass as third argument "separator" the value "-" if you need the language-code

Example: format('en', 'US', { separator: '-' })

Args: {"language":"en","iso":"US","options":{"separator":"-"}}

en-US

Fn: format (with type: language-code)

You can pass a third argument "type" if you need the language-code

Example: format('en', 'US', { type: 'language-code' })

Args: {"language":"en","iso":"US","options":{"type":"language-code"}}

en-US

Fn: format (with type: locale)

You can pass a third argument "locale" if you need the locale format

Example: format('en', 'US', { type: 'locale' })

Args: {"language":"en","iso":"US","options":{"type":"locale"}}

en_US

Fn: ISO

A function that returns the ISO data for a given ISO code

Example: ISO('US')

Args: {"ISO":"US"}

{
  "languages": [
    "en"
  ],
  "name": "United States",
  "original": "United States"
}

Fn: ISO ("languages")

A function that returns the ISO data for a given the country code Get the languages for a given ISO code

Example: ISO('BE', 'languages')

Args: {"ISO":"BE","type":"languages"}

{
  "languages": [
    "nl",
    "fr",
    "de"
  ],
  "name": "Belgium",
  "original": "België"
}

Fn: ISO ("names")

A function that returns the ISO data for a given the country code Get the name of the country for a given ISO code

Example: ISO('BE', 'names')

Args: {"ISO1":"BE","type":"names"}

{
  "languages": [
    "nl",
    "fr",
    "de"
  ],
  "name": "Belgium",
  "original": "België"
}

Fn: ISO ("riginal")

A function that returns the ISO data for a given the country code Get the original name of the country for a given ISO code

Example: ISO('BE', 'original')

Args: {"ISO1":"BE","type":"original"}

België

Fn: ISO ("language-code")

A function that returns the ISO data for a given the country code Get the language code for a given ISO code

Example: ISO('BE', 'language-code')

Args: {"ISO1":"BE","type":"language-code"}

["nl-BE","fr-BE","de-BE"]

Fn: getCountryData

Get country data by a given a language code (e.g. "es-ES")

Example: getCountryData('es-ES')

Args: {"languageCode":"es"}

{
  "languages": [
    "es",
    "eu",
    "ca",
    "gl",
    "oc"
  ],
  "name": "Spain",
  "original": "España"
}

Fn: getCountry

Get country data by a given the country name (e.g. "Italy") or country original name (e.g. "Italia")

Example: getCountry('Italia')

Args: {"name":"Italia"}

{
  "languages": [
    "it"
  ],
  "name": "Italy",
  "original": "Italia",
  "code": "IT"
}
